MQTT Monitor Documentation

Overview

MQTT Monitor is a custom Home Assistant integration that provides real-time monitoring of MQTT topics and messages. It allows you to filter, view, and analyze MQTT data directly within Home Assistant.

Features

  • Real-time MQTT message monitoring.
  • Customizable topic filtering.
  • Supports logging and historical message storage.
  • Display messages with a clean and responsive UI.
  • Easily integrated with Home Assistant dashboards.

Installation

Manual Installation

  1. Download the mqtt_monitor custom integration and associated files from the repository.
  2. Navigate to your Home Assistant configuration directory: 
    /config/
  3. Create or locate the custom_components directory: 
    mkdir /config/custom_components
  4. Place the mqtt_monitor folder into the custom_components directory.
  5. Additionally, copy the following files to your /config/www directory:
    • mqtt_monitor.html
    • mqtt_monitor.png
  6. Restart Home Assistant.

Directory Structure

/config
├── custom_components
│   └── mqtt_monitor
│       ├── __init__.py
│       ├── manifest.json
│       └── sensor.py
├── www
│   ├── mqtt_monitor.html
│   └── mqtt_monitor.png
├── configuration.yaml

Adding an MQTT Monitor Dashboard

You can embed mqtt_monitor.html directly into your Home Assistant Lovelace dashboard using an iframe card. This enables you to view real-time MQTT data without leaving Home Assistant.

1. Confirm File Location

Ensure mqtt_monitor.html and mqtt_monitor.png are placed in your /config/www folder, making them accessible at /local/mqtt_monitor.html and /local/mqtt_monitor.png.

2. Add an iframe Card

To display the file, add an iframe card referencing /local/mqtt_monitor.html (with an optional version parameter ?v=57 for cache-busting). Here are two methods:

Option A: Using the UI Editor

  1. In Home Assistant, go to Settings > Dashboards.
  2. Click Add Card, search for Manual.
  3. Paste the following YAML into the card editor:
type: iframe
url: /local/mqtt_monitor.html?v=57
title: MQTT Monitor

Click Save to add the card.
